# Screenshots Directory

Place your database viewer screenshots here:

- `database_viewer.png` - Main database viewer interface showing all tables
- `tasks_table.png` - Sample tasks table view
- `projects_overview.png` - Projects overview with sections

To capture screenshots:
1. Run `python view_database.py`
2. Open http://localhost:8000 in your browser
3. Take screenshots of the interface
4. Save them in this directory
